Study Hangman

## Project Description

Study Hangman is a browser-based Hangman game created using HTML, SCSS, TypeScript, and JSON.

The goal of the game is to correctly guess a hidden word before reaching the maximum number of incorrect guesses. The game randomly selects a word, category, and hint from a JSON file each time a new game starts.

## Features

* Random word selection from a JSON file
* Multiple categories
* Hint system
* Tracks incorrect guesses
* Displays hangman images
* Win and lose messages
* Play Again button
* Responsive design
* SCSS styling with mixins
* TypeScript game logic

## How to Play

1. Read the category and hint.
2. Enter one letter at a time.
3. Correct letters will appear in the word.
4. Incorrect letters increase the hangman progress.
5. Guess the entire word before reaching 6 wrong guesses.
6. Click Play Again to start a new game.
