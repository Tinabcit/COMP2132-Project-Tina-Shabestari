# COMP2132-Project-Tina-Shabestari

**Hangman Game Project Description**

This project is a browser-based Hangman game developed using HTML, CSS, and JavaScript. The game randomly selects a word and a corresponding hint from a predefined collection. The player's objective is to correctly guess the hidden word by selecting letters from the alphabet.

When a correct letter is chosen, the game reveals all occurrences of that letter within the word. If an incorrect letter is selected, a new part of the hangman graphic is displayed. The player is allowed up to six incorrect guesses before the game ends.

The game tracks guessed letters and disables each letter after it has been selected to prevent duplicate guesses. The game continuously checks for win and loss conditions. A player wins by correctly revealing every letter in the word and loses when the maximum number of incorrect guesses is reached.

Once the game is over, the application displays a message indicating whether the player won or lost and provides a "Play Again" button. Selecting "Play Again" resets the game state, re-enables all letter buttons, chooses a new random word and hint, and allows the player to start a new game.

This project demonstrates the use of JavaScript arrays, functions, event handling, conditional statements, loops, random number generation, and DOM manipulation to create an interactive web application.
